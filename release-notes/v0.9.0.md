# x-agentic-workflow v0.9.0

This release makes the desktop workspace flow safer by scoping browser-desktop
sessions to the active project path.

## Highlights

- **Project-scoped desktop sessions**: `xaw desktop` now maps each active
  workdir to its own session directory under the local sessions root.
- **Filtered session list**: the desktop session list only shows conversations
  for the active project.
- **Switch-back restore**: switching back to a previous project restores that
  project's own session list.
- **Path-specific namespace**: projects with the same folder name under
  different parent paths receive distinct session storage.

inside `xaw desktop`; CLI and TUI session behavior remain unchanged.
