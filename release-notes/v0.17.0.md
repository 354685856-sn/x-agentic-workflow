# cat-agentic v0.17.0

This release renames the product from `x-agentic-workflow` to `cat-agentic`
while keeping the `xaw` and `x-agentic-workflow` CLI commands as compatibility
aliases. The Python import module and existing local configuration directory
also remain unchanged to preserve user data.

## Desktop

- Aligns the left navigation and composer with the current product direction.
- Adds a real local scheduled-task store and desktop-process scheduler.
- Shows current Git branch, worktree, changed files, and diff in the inspector.
- Lists Git Worktrees, creates a new branch Worktree, and switches projects.
- Keeps destructive Worktree deletion out of the UI.

## Distribution

- Adds the primary `cat-agentic` CLI command.
- Renames the macOS preview app to `Cat Agentic.app`.
- Updates the macOS launcher, DMG build, smoke, and signing paths.

## Compatibility

- `xaw` remains available.
- `x-agentic-workflow` remains available.
- `x_agentic_workflow` remains the Python import package.
- `~/.x-agentic-workflow` remains the local configuration and session location.

## Validation

- `.venv/bin/python -m pytest`
- `.venv/bin/python -m ruff check src tests`
- `.venv/bin/python -m mypy src/x_agentic_workflow`
- `.venv/bin/cat-agentic --version`
- `.venv/bin/xaw --version`
- `.venv/bin/x-agentic-workflow --version`
