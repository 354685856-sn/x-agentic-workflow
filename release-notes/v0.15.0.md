# x-agentic-workflow v0.15.0

This release adds scoped composer draft recovery and fixes rapid new-session id
collisions discovered during browser validation.

## Highlights

- **Refresh-safe drafts**: composer text survives browser refresh.
- **Project/session scoping**: drafts are keyed by active workdir and session id,
  preventing unrelated sessions from sharing text.
- **Session return**: switching away and reopening a session restores its draft.
- **Successful-send cleanup**: accepted sends remove the persisted draft.
- **Bounded local storage**: drafts are limited to 64 KiB and storage failures
  do not block the composer.
- **No attachment persistence**: file bodies remain in memory only.
- **Unique rapid sessions**: new session ids now include microseconds, preventing
  same-second collisions.
