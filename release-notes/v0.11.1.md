# x-agentic-workflow v0.11.1

This patch release aligns the desktop home and settings UI with the current
product standard and removes navigation entries for features that are not yet
implemented.

## Highlights

- **Focused home screen**: replaces the dashboard-style stats panel with an
  empty-session screen and bottom composer.
- **Provider settings alignment**: keeps settings centered on the working
  provider-management flow: save metadata, test connection, and review provider
  candidates.
- **No fake navigation**: removes unimplemented sidebar/settings entries such as
  scheduled tasks, plugins, agents, skills, memory, computer use, and trace.
- **Existing controls preserved**: new session, provider save/test, project
  switch, project validation, and session/file-ledger behavior remain intact.
