# x-agentic-workflow v0.10.0

This release adds the first desktop file-change workspace slice: a live ledger
for files changed by `write_file` tool calls and a latest-diff viewer.

## Highlights

- **File-change ledger**: desktop state now records successful `write_file`
  tool results with path, status, summary, and diff metadata.
- **Latest diff viewer**: the right inspector renders the latest unified diff
  from tool metadata.
- **Cleaner inspector output**: static placeholder output rows were replaced by
  real file-change and diff panels.
- **Scoped lifecycle**: file-change state resets when starting a new desktop
  chat or switching projects.

## Confidence

Medium-high for the scoped file-ledger slice.

## Scope Risk

Low-to-moderate. The feature consumes existing `write_file` metadata and does
not change file-tool semantics, command approval, or provider behavior.

## Not Included

- Multi-file diff selection.
- Persisted file-change ledger per session.
- Non-`write_file` file mutation detection.
- Native visual diff renderer.
