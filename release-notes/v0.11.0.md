# x-agentic-workflow v0.11.0

This release makes the desktop file-change ledger durable per session and adds
selection for older file diffs in the inspector.

## Highlights

- **Persisted file ledger**: desktop `write_file` events are saved into the
  active session JSON under `file_changes`.
- **Session restore**: opening a prior desktop session restores its file-change
  ledger and defaults the diff viewer to the most recent change.
- **Multi-file diff selection**: file-change rows are selectable, with a
  dedicated `/api/diff/select` desktop endpoint and selected diff state.
- **Compatibility**: older session JSON without `file_changes` still loads, and
  normal message saves preserve file-change metadata.

## Confidence

Medium-high for the scoped desktop session persistence and diff-selection slice.

## Scope Risk

Low-to-moderate. The change extends session JSON metadata and desktop state, but
does not change file-tool execution, command approval, provider behavior, or the
CLI/TUI conversation loading contract.

## Not Included

- Native visual side-by-side diff renderer.
- Non-`write_file` mutation detection.
- Cross-session search over file changes.
